## Version History

- **6.2.0** (2026-04-01): Derived artifacts (technical-design.md, visual-design.md) auto-generated by `/rs-spec`
- **6.1.0** (2026-04-01): User-defined phases, axioms 7-9, methodology distinction
- **6.0.0** (2026-03-28): Four-skill architecture, `npx skills` installation
- **5.x**: Plugin-based distribution
- **4.x**: CLI with auto-detection and template engine
- **3.x**: Manual prompt workflow
